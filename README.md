# ブロックス (Blokus)

iPhone / スマホで遊べるブロックスのボードゲームです。
外部ライブラリ不要、HTMLファイル1つで動作します。

## 遊び方

### ゲームモード

| モード | 説明 |
|--------|------|
| 1人 vs CPU×3 | あなた（赤）vs コンピュータ3体 |
| 4人対戦 | ローカルマルチプレイ（1台で交代） |

### 操作方法

1. 画面下のピース一覧からピースをタップして選択
2. **↻** で回転、**↔** で反転
3. ボード上をタップして配置
4. 置けない場合は **パス** ボタンを押す

### ルール

- 各プレイヤーは21種類のピース（1〜5マス）を持つ
- 最初のピースは自分のコーナー（角）に置く
- 2手目以降は、自分の既存ピースと **角で接する** 位置に置く
- 自分のピース同士は **辺で接してはいけない**
- 全員がパスしたらゲーム終了

### スコア

- 置けなかったピースのマス数がマイナス点
- 全ピース配置で **+15ボーナス**
- スコアが最も高いプレイヤーの勝ち

## 実行方法

```
blokus/index.html をブラウザで開く
```

サーバー不要。ファイルをダブルクリック、またはiPhoneにAirDrop等で送って開くだけで動作します。

## 技術仕様

- HTML / CSS / JavaScript（単一ファイル）
- Canvas API でボード描画
- タッチ操作対応（iPhone Safari最適化）
- CPU AI: 大きいピース優先＋角確保の貪欲アルゴリズム
